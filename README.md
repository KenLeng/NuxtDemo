# NuxtDemo

需新增.env 設定環境變數(可複製.env.example)

80 port - proxy server port
3001 port - node server port

## Build Setup

```bash
# 安裝node相關套件
$ npm install

# 開發模式，預設3001 port
$ npm run dev

# 編譯nuxt程式
$ npm run build
# 起server 預設3001 port
$ npm run start

```

For detailed explanation on how things work, check out [Nuxt.js docs](https://nuxtjs.org).
